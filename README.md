# 環境設定

1. Node.jsのバージョン `18.16.0`をインストールしてください。

2. 以下コマンドを実行し、npm packageをインストールしてください。

```
yarn
```

3. 以下コマンドを実行し、開発サーバーが立ち上がることを確認してください。

```
yarn dev
```
